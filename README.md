# Loading Animation

Projeto desenvolvido para praticar animações utilizando HTML e CSS através da criação de uma loading animation moderna e dinâmica.

## 🚀 Objetivo

O objetivo deste projeto foi aprimorar conhecimentos em animações CSS, efeitos visuais e construção de componentes utilizados em interfaces web modernas.

## 🛠 Tecnologias utilizadas

- HTML5
- CSS3

## ✨ Funcionalidades

- Animação de carregamento
- Transições suaves
- Design minimalista
- Efeitos visuais modernos
- Estrutura simples e organizada

## 📚 Conceitos praticados

- CSS Animations
- Keyframes
- Transitions
- Responsividade
- Estilização visual
